# inssman_tts
fixing vampi tts and voiceforge
# how to get
you need some browser which supports inssman like google chrome or microsoft edge
you can install or open [inssman here](https://www.inssman.com/app/)
you can also get [kiwi browser]([https://github.com/kiwibrowser/src.next) if your android user (isnt supported for ios)
i recommend arm version of kiwi browser
# how to import
you need to go on inssman app, then you will see the button "import", then import rules.json
[download the rules.json](https://grapzmen-goanimation.github.io/inssman_tts/install/files/json/download.html)
